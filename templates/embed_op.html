Embedded Ops

The Embedded Ops Unit specializes in developing electronic solutions by integrating hardware and software to create systems designed for specific tasks. The unit's focus includes programming, system bring-up and re-engineering/remanufacturing of embedded systems. Also, the Ops would explore innovative frontiers in processor architecture design.

Embedded systems are now central to everyday functionality, powering devices ranging from microwave ovens and washing machines to smartphones, industrial machinery, and avionics. Their ubiquity is set to expand dramatically with the mass adoption of the Internet of Things (IoT), enabling smart systems like intelligent cities, advanced electricity grids, and Industry 4.0 applications supported by next-generation wireless networks.

Recognizing the critical importance of embedded systems in electronics and smart solutions, the Embedded Ops Unit aims to develop, cultivate, and deploy embedded systems skills to address challenges with electronic systems in Gutenkraft projects.

The unit's focus areas include:

Hardware Components:

  • Single board microcontrollers/computers (SBM/C) (Arduino)
  • System-on-chip (SoC) computers (Raspberry Pi, Beaglebones, Nvidia Jetson boards and custormized SoCs)
  • Application specific integrated chips (ASICs) boards
  • Field programmable gate array (FPGA) boards

Architecture and Operating System:

  • Open-source processor IP designs ( RISC-V instruction set architecture)
  • Portable Embedded Linux kernels, (specifically Yocto custom builds)
  • Real-Time Operating Systems (FreeRTOS, Zephyr and other promising open-source RTOS)

Programming Languages and Tools:

  • Embedded C language and compilers
  • Rust language
  • Qt framework
  • Python programming language
  • Hardware description languages (Verilog and VHDL)
  • Robot operating system (ROS) software framework

The ultimate goal is to develop reliable and cost-effective electronic solutions across various domains while reducing e-waste.
